Project Title and SDG Goal: Healthy Food & Calorie Tracker. Our goal is to help people to mange their food and track their calories.
Project Description: The Healthy Food & Calorie Tracker is a mobile application that directly contributes to Sustainable Development Goal 3: Good Health and Well-Being by assisting users in their journey towards making healthier food choices and leading a balanced lifestyle. It enables the user to keep a record of his or her daily calorie consumption, check the nutritional content of the food that is being consumed, and acquire the necessary skill of eating right.
Installation/Setup: 
Step 1: Install a C++ Compiler
Make sure g++ is installed on your computer.
Windows: Install Code::Blocks or MinGW
macOS: Run xcode-select --install
Linux: Run sudo apt install g++

Step 2: Create the Program File
Open Notepad or any text editor
Copy the C++ code
Save the file as:
food_menu.cpp

Step 3: Open Terminal or Command Prompt
Go to the folder where food_menu.cpp is saved
Right-click → Open Terminal / Command Prompt here

Step 4: Compile the Program
Type this command and press Enter:
g++ food_menu.cpp -o food_menu

Step 5: Run the Program
Windows:
food_menu
macOS / Linux:
./food_menu

Step 6: Use the Program
Choose a food category (1–5)
Select a food item
View its calorie details
Choose 6 to exit

Step 7: Exit the Program
Type 6 in the menu to close the program.

Usage Instructions: To use the Healthy Food & Calorie Tracker you need to download the app and right after you download you can check the calories of the food by searching the names or searching the calories to pop-up which food contaions a calories.
